WORKS

Created interactive games such as Breakout Game, Bat Eating Butterfly and Snake Game in Scratch, applying logic and creativity to engage players

Breakout Game

The game is a single-player version of Pong, where the player controls a paddle at the bottom of the screen to bounce a ball upward to break a wall of bricks.

Bat Eating Butterfly

Bat Eating Butterfly is a simple game created in Scratch (a visual programming language) where a bat chases and eats butterflies. Since Scratch allows users to create and share their own projects, the exact gameplay may vary depending on the creator

Snake

The Snake Game is a classic arcade-style game where the player controls a growing snake that moves around the screen to eat food while avoiding collisions. It originated as a blockade-style game in the 1970s and gained massive popularity with versions on Nokia mobile phones in the late 1990s.
